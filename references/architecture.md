# 多智能体自动化系统 — 架构设计参考

## 四层闭环架构

```
┌─────────────────────────────────────────────────────────┐
│                    感知层 · 多模态输入网关                   │
│  飞书群聊(WebSocket)  │  文件上传(图/PDF)  │  网页链接(SPA)   │
│       格式自动判别  │  自适应提取策略  │  环境依赖自探测      │
└──────────────────────────┬──────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────┐
│               推理层 · 长链任务规划 + 动态路由               │
│  格式判别→文本提取→字段解析→类别推断→命名匹配→封装导出→推送  │
│           三级容错：DeepSeek-V4 → Qwen3-8B → V4-Free      │
└──────────────────────────┬──────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────┐
│             执行层 · 可复用技能库 + 跨会话记忆               │
│  工具链编排  │  流程固化为 Skill  │  持久记忆跨会话复用       │
└──────────────────────────┬──────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────┐
│          沉淀层 · 群聊知识蒸馏引擎 (核心创新)               │
│  语义聚类  │  去噪摘要  │  结构化改写  │  月度全量回溯        │
└──────────────────────────┬──────────────────────────────┘
                           ▼
              ┌─────────────────────────┐
              │     组织知识库           │
              │  检索 · 追溯 · 复用      │
              └─────────────────────────┘
                           │
         闭环反馈 ◄─────────┘  知识反哺决策
```

## 三级模型容错链

```
主力推理节点 (DeepSeek-V4-Flash, SiliconFlow)
    │  失败 ↓ <1s 切换
备用推理集群 (Qwen3-8B, SiliconFlow)
    │  失败 ↓ <1s 切换
兜底免费节点 (DeepSeek-V4-Flash-Free, Zenmux, 1M ctx)
    │
    ▼  始终可用
```

## 技能模块拓扑

```
multi-agent-automation (系统级)
├── invoice-pdf-renamer      — 发票OCR→命名→归档
├── web-article-to-word       — 网页→结构化docx
├── ocr-and-documents         — 通用OCR引擎
└── [群聊蒸馏引擎]             — WebSocket→知识条目 (内置)
```

## 数据流

```
用户输入 (飞书消息)
    │
    ▼
感知层: 格式判别 → 策略选择
    │
    ▼
推理层: 任务规划 → 模型路由 → 容错切换
    │
    ▼
执行层: 工具调用 → 链式编排 → 结果封装
    │
    ▼
沉淀层: 知识提取 → 结构化 → 归档
    │
    ▼
用户输出 (飞书消息/文件) + 知识库写入
```
